# JavaScript & TypeScript Data Structures and Algorithms Workbook

Personal Data Structures and Algorithms (DSA) workbook. This repository serves as a practical playground where I implement foundational computer science concepts, solve algorithmic challenges, and master core interview patterns using JavaScript and TypeScript.

The codebase tracks my journey through various online courses, competitive programming platforms, and targeted practice roadmaps.

---

## 🚀 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) and [Yarn](https://yarnpkg.com/) installed on your machine.

### Development Commands

Use the following scripts to manage the project compilation and workspace:

| Command                     | Description                                                     |
| --------------------------- | --------------------------------------------------------------- |
| `yarn install`              | Install all project dependencies                                |
| `yarn run watch`            | Automatically watch files for changes and incrementally compile |
| `yarn run build`            | Manually compile all TS files to JS                             |
| `bash scripts/compile.bash` | Run the custom automation script to build all modules           |
